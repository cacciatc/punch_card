# PunchCard

[![Build Status](https://travis-ci.org/cacciatc/punch_card.svg?branch=master)](https://travis-ci.org/cacciatc/punch_card)

Create punch cards in various formats and save as PNGs.

## Supported Cards and Formats

* IBM 5081
	* ASCII 8-bit (WIP)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'punch_card'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install punch_card

## Usage

```ruby
require 'punch_card'

include PunchCard

# create a card
card = PunchCard.create('PUNCHCARD punchcard', Formats::IBM5081, Encodings::ASCII8)

# save to file
card.to_txt('card.txt')
```
