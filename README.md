# PunchCard

Create punch cards in various formats and save as PNGs.

## Supported Cards and Formats

* IBM 5081
	* ASCII 8-bit (WIP)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'punch_card'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install punch_card

## Usage

```ruby
require 'punch_card'

include PunchCard

# create a card containing the letters A, B, C
card = Encodings::ASCII8.encode('ABC', Formats::IBM5081)

# comment
card.comments = 'ABC'

# save to file
card.to_txt('card.txt')
```
